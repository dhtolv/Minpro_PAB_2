## જ⁀➴ Deskripsi Aplikasi •
Aplikasi Booking Lapangan ini adalah aplikasi sederhana yang digunakan untuk melakukan pemesanan lapangan secara online. Pengguna cukup mengisi data diri dan memilih lapangan serta tanggal yang diinginkan, lalu booking akan tersimpan ke dalam sistem. Data booking disimpan menggunakan Supabase sehingga dapat dikelola secara online. Aplikasi ini juga dilengkapi dengan fitur dark mode untuk meningkatkan kenyamanan pengguna saat menggunakan aplikasi.

## જ⁀➴ Fitur Aplikasi •

| Fitur | Kegunaan |
|-----|---|
| **Registrasi Akun** | Membuat akun baru dengan menginputkan email, password, dan konfirmasi password. Fitur register digunakan untuk membuat akun baru agar pengguna dapat mengakses aplikasi dan melakukan pemesanan lapangan.|
| **Login** | Masuk ke aplikasi menggunakan email dan password. Fitur login memungkinkan pengguna masuk ke dalam aplikasi menggunakan email dan password yang telah terdaftar di Supabase.|
| **Halaman Utama** | Menampilkan semua data booking lapangan yang telah dibuat pengguna.|
| **Menambah Booking** | Pengguna dapat menambahkan data booking lapangan dengan mengisi form yang tersedia seperti nama pemesan, nomor telepon, jenis lapangan, dan tanggal. |
| **Mengedit Booking** | Data booking yang sudah dibuat dapat diperbarui apabila terdapat kesalahan atau perubahan data. |
| **Menghapus Booking** | Pengguna dapat menghapus data booking yang tidak diperlukan lagi. |
| **Tema Aplikasi** | Aplikasi menyediakan fitur dark mode yang dapat mengubah tampilan aplikasi menjadi mode gelap. |

## જ⁀➴ Widget yang Digunakan •

| Widget | Keterangan |
|-----|---|
| **MaterialApp** | Widget utama yang digunakan sebagai root aplikasi Flutter untuk mengatur tema, routing halaman, dan konfigurasi dasar aplikasi.|
| **Scaffold** | Digunakan sebagai kerangka utama halaman aplikasi yang menyediakan struktur seperti AppBar, body, dan FloatingActionButton.|
| **AppBar** | Widget yang berfungsi sebagai bagian header pada halaman aplikasi untuk menampilkan judul halaman dan beberapa tombol aksi seperti dark mode dan logout.|
| **SafeArea** | Widget yang berfungsi sebagai bagian header pada halaman aplikasi untuk menampilkan judul halaman dan beberapa tombol aksi seperti dark mode dan logout. |
| **Padding** | Digunakan untuk memberikan jarak atau ruang di sekitar widget agar tampilan lebih rapi. |
| **SizedBox** | Widget yang digunakan untuk memberikan jarak antar komponen atau menentukan ukuran tetap suatu widget. |
| **ConstrainedBox** | Digunakan untuk membatasi ukuran maksimal atau minimal suatu widget agar tampilan tetap proporsional di berbagai ukuran layar. |
| **Stack** | Digunakan untuk menumpuk beberapa widget sehingga satu widget dapat berada di atas widget lainnya, misalnya untuk membuat gambar sebagai background.|
| **ClipRRect** | Digunakan untuk memotong bentuk widget dengan sudut melengkung sehingga tampilan menjadi lebih menarik.|
| **Form** | Digunakan sebagai pembungkus kumpulan input form untuk memudahkan proses validasi data. |
| **CircleAvatar** | Widget yang digunakan untuk menampilkan ikon berbentuk lingkaran, biasanya digunakan untuk menampilkan huruf awal dari nama pengguna. |
| **Image** | Widget yang digunakan untuk menampilkan gambar seperti banner pada halaman login, register, dan halaman utama aplikasi. |
| **Text** | Digunakan untuk menampilkan teks pada aplikasi seperti judul halaman, label, dan informasi lainnya. |
| **Icon** | Widget yang digunakan untuk menampilkan ikon pada berbagai komponen UI seperti pada tombol atau input field. |
| **TextFormField** | Widget input yang digunakan untuk menerima data dari pengguna seperti email, password, nama, dan nomor telepon. |
| **TextEditingController** | Digunakan untuk mengontrol dan mengambil nilai dari input pada TextFormField. |
